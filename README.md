# Description

This initiative is about using the Internet Protocol (IP) suite for deep space, as described in this [IETF draft](https://datatracker.ietf.org/doc/draft-many-deepspace-ip-architecture/), based on a reassessment discussed in this [IETF draft](https://datatracker.ietf.org/doc/draft-many-deepspace-ip-assessment/). It is an alternative to the use of the [Bundle Protocol](https://datatracker.ietf.org/doc/rfc9171/) to create a network in deep space.
